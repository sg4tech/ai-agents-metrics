# Marketing Best Practices — External Canon

Reference library of external marketing best practices. This is **not** project-specific to `codex-metrics` — it is the broader industry canon, organized for scanning and selective deep-dives.

## How to use

- **Not mandatory reading.** This folder is optional reference for GTM, positioning, messaging, and distribution decisions. Consult the relevant file when a specific decision calls for it.
- **Names, not summaries.** Each file lists the canonical authors/works and the core frameworks they introduced. Read the source if the framework becomes load-bearing for a decision — this library does not substitute for the originals.
- **Project-specific application** lives elsewhere: `docs/private/product-strategy.md`, `oss/docs/product-framing.md`.

## Top 10 books for a lifetime

A compressed shortlist. If you only read ten books on marketing, read these.

1. **April Dunford** — *Obviously Awesome* (positioning as a teachable craft)
2. **David Ogilvy** — *Ogilvy on Advertising* (timeless copywriting and research discipline)
3. **Seth Godin** — *This Is Marketing* (permission marketing, smallest viable market)
4. **Robert Cialdini** — *Influence: The Psychology of Persuasion* (seven principles of influence)
5. **Al Ries & Jack Trout** — *Positioning: The Battle for Your Mind* (the original category framing)
6. **Geoffrey Moore** — *Crossing the Chasm* (tech adoption lifecycle; the chasm)
7. **Gabriel Weinberg & Justin Mares** — *Traction* (19 channels; Bullseye framework)
8. **Eugene Schwartz** — *Breakthrough Advertising* (five awareness levels; mass desire)
9. **Alexander Osterwalder** — *Value Proposition Design* (customer jobs, pains, gains; the canvas)
10. **Claude Hopkins** — *Scientific Advertising* (the original data-driven advertising, 1923)

**Honourable mentions:** Andrew Chen — *The Cold Start Problem* (network effects); Madhavan Ramanujam — *Monetizing Innovation* (price before product); Philip Kotler — *Marketing Management* (the textbook canon).

**Aperitif:** David Ogilvy — "We Sell, or Else" speech. Fifteen minutes. The clearest statement that marketing exists to produce measurable results, not awards. Equivalent calibration to Horowitz's "Good PM / Bad PM" memo.
